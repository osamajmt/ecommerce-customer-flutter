# Ecommerce Customer App (Flutter)

Customer mobile application for a multi-role ecommerce system.

## Features
- User registration & login
- Product browsing
- Cart & checkout
- Order history
- API integration

## Tech Stack
- Flutter
- Dart
- REST API
- GetX
